# OPEN-ENDED LAB

## Title

**Design and Development of a Containerized Full-Stack Application using
C# and Docker**

------------------------------------------------------------------------

## Objective

To enable students to analyze, design, and implement a real-world
application using:

-   Object-Oriented Programming\
-   Visual Programming (WinForms / ASP.NET Core)\
-   Database Integration\
-   REST APIs\
-   Docker-based Deployment

------------------------------------------------------------------------

## Problem Statement (Open-Ended)

Students are required to identify a real-world problem and develop a
complete software solution that includes:

-   A user interface (desktop or web)\
-   Backend logic using C#\
-   Database integration\
-   API interaction (optional but encouraged)\
-   Deployment using Docker containers

------------------------------------------------------------------------

## Example Problem Domains

(Students can choose or propose their own)

-   Student Management System\
-   Inventory Management System\
-   Hospital/Clinic Management System\
-   Online Booking System\
-   Task/Project Management Tool\
-   E-commerce mini system

------------------------------------------------------------------------

## Minimum Requirements

### 1. Application Development

-   Use C# (.NET / ASP.NET Core / WinForms)\
-   Apply OOP principles:
    -   Classes & Objects\
    -   Inheritance\
    -   Polymorphism\
    -   Encapsulation

### 2. User Interface

-   Functional UI (WinForms or Web)\
-   Use of controls (forms, buttons, menus, etc.)\
-   Event-driven programming

### 3. Database Integration

-   Use SQL Server / any relational DB\
-   Perform CRUD operations:
    -   Create\
    -   Read\
    -   Update\
    -   Delete

### 4. API Integration (Optional but High Weightage)

-   Create OR consume REST APIs\
-   Display API data in UI

### 5. Docker Integration (Mandatory)

Students must: - Create a Dockerfile\
- Build Docker image for their application\
- Run application inside a container\
- (Bonus) Use Docker Compose for multi-container setup (App + DB)

### 6. Additional Features (Encouraged)

-   Authentication/Login system\
-   File handling\
-   Exception handling\
-   Logging/Debugging\
-   Clean UI/UX

------------------------------------------------------------------------

## Expected Learning Outcomes

After completing this lab, students will be able to:

-   Apply object-oriented principles in real applications\
-   Design and implement complete software systems\
-   Deploy applications using Docker\
-   Work independently on open-ended problems\
-   Integrate multiple technologies into a single solution
