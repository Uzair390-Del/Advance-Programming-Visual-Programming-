#  Assignment #1: AI-Based Application Development in .NET

##  Objective
The goal of this assignment is to help students understand the **complete lifecycle of building an AI-based application** using .NET — from concept to deployment.

Students will:
- Understand theoretical concepts of AI integration  
- Build a working AI-powered mini project  
- Document their work professionally  
- Use industry tools like GitHub and Docker  

---

##  Title
**Assignment #1 – AI-Based Application Development in .NET (Theory)**

#  Part 3: Docker Requirement

Students must:

- Create a **Dockerfile** for their application  
- Explain:
  - Base image  
  - Build process  
  - Run command  

###  Example Dockerfile
```dockerfile
FROM mcr.microsoft.com/dotnet/sdk:7.0
WORKDIR /app
COPY . .
RUN dotnet build
CMD ["dotnet", "run"]
```
