#  Assignment #2: Full-Stack Application Development using .NET

##  Objective

This assignment focuses on applying the complete software development process (CLO-02) to build a real-world data-driven application using .NET technologies.

Students will:
- Understand system design and development lifecycle  
- Develop a multi-layered application  
- Apply database, file handling, and API concepts  
- Practice testing, debugging, and documentation
